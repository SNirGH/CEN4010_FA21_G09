Animation

Drawing

Along with animation, I do enjoy drawing things for myself. Taking an idea from my head and realizing it by drawing it on a blank canvas is something I really love doing. It allows me to create unique characters that I have imagined and be able to see it in real life.

Programming

Programming is a lot like animation and drawing for me: taking an idea and implementing it. In this case, the implementation involves writing programs that can become software, games, etc.
